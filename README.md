# Next.js Notes App 📝

A full-stack Notes Application built using Next.js and MongoDB. This project was created to revise and strengthen core full-stack concepts including API routes, database integration, and CRUD operations.

---

## 🚀 Features

* ✅ Create notes
* ✅ Fetch and display notes
* ✅ Update notes (edit functionality)
* ✅ Delete notes
* ✅ Responsive UI with card layout
* ✅ Real-time UI updates after actions

---

## 🛠️ Tech Stack

* **Frontend:** Next.js (React), Tailwind CSS
* **Backend:** Next.js API Routes
* **Database:** MongoDB (Mongoose)
* **State Management:** React Hooks (useState)

---

## 📂 Project Structure

```
app/
 ├── api/notes/        # API routes (GET, POST, PUT, DELETE)
 ├── page.js           # Main page
components/
 ├── NotesClient.jsx   # Frontend logic & UI
lib/
 ├── db.js             # Database connection
models/
 ├── Note.js           # Mongoose schema
```

---

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/abhishek-kr01/nextjs-notes-app
```

2. Install dependencies

```bash
npm install
```

3. Add environment variables in `.env`

```env
MONGODB_URI=your_mongodb_connection_string
```

4. Run the app

```bash
npm run dev
```
